## ADDED Requirements

### Requirement: Comparison view
The system SHALL provide a way for users to select multiple restaurant cards to compare side-by-side or in an expanded view.

#### Scenario: User compares restaurants
- **WHEN** the user selects "Compare" on two or more restaurant cards
- **THEN** an overlay or separate screen appears highlighting the differences (e.g., price, distance, rating)

### Requirement: Highlighted recommendation
The system SHALL highlight a single "Best Match" restaurant based on the search relevance and distance.

#### Scenario: Viewing search results
- **WHEN** the results list is displayed
- **THEN** one card is visually distinguished as the "Best Match"
